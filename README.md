# Chrome-Transfer

浏览器划词翻译插件。

## 简介

Chrome-Transfer 是一个轻量的浏览器扩展，用于通过划词或选中文本快速获取翻译结果，方便阅读外文网页内容。

## 功能

- 划词/选中翻译
- 弹出结果窗口（非侵入式）
- 支持复制翻译结果

## 安装（开发者模式）

1. 克隆或下载本仓库到本地：

   git clone https://github.com/Miles1994/Chrome-Transfer.git

2. 打开 Chrome 扩展管理页：chrome://extensions
3. 打开右上角的“开发者模式”开关
4. 点击“加载已解压的扩展程序”，选择仓库根目录

> 如果此项目已发布到 Chrome 网上应用店，请直接从商店安装以获得自动更新。

## 使用方法

1. 在网页上选中需要翻译的文字。
2. 使用扩展提供的快捷方式或右键菜单（若存在）触发翻译，或点击浏览器工具栏中的扩展图标查看翻译结果。
